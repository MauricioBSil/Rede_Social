# Objetivo

O projeto se trata de um estudo sobre desenvolvimento para redes sociais, com foco no aprendizado e domínio das linguagens HTML, CSS e Javascript.
O modelo que foi usado como inspiração foi o Instagram, mas durante o desenvolvimento surgiram ideias de como diferenciar a versão.

# Tecnologias Utilizadas

HTML e CSS, e futuramente será implementado Javascript.
